# PocketForge OS

PocketForge OS is an open-source Fedora Atomic-based gaming operating system built for PC gaming handhelds.

It aims to provide a console-like experience for devices such as the ASUS ROG Ally, ROG Ally X, Lenovo Legion Go, Steam Deck, and other x86 gaming handhelds, while keeping the flexibility of Linux.

## Goals

- Handheld-first gaming experience
- Atomic updates with rollback support
- KDE Plasma desktop mode
- Steam and Gamescope gaming support
- Device profiles for popular handhelds
- Beautiful custom control center
- Performance and battery profiles
- Open-source and community-driven

## Status

PocketForge OS is currently in early alpha development.

Do not install it on your main device yet unless you know how to recover or rollback your system.
